⮐ Voltar

Sistema em microsserviços para cadastro de vendas, controle de parcelas e geração de relatórios (2023.1)

Sobre o Projeto

Trabalho desenvolvido para o Projeto Integrador no 3º semestre da Fatec, tendo como cliente a empresa Pro4Tech, que atua na transformação de empresas físicas para o mundo digital.

Contribuições

Neste projeto, o objetivo era criar um software que lidasse com sistemas de gestão um tipo novo de projeto em que precisávamos lidar corretamente com valores monetários, gerenciamento de datas e status de pendência ou pagamento. Além disso, foi o primeiro projeto em que utilizamos React na arquitetura do frontend, e Java na arquitetura do backend. Atuei tanto no frontend quanto no backend, por isso consegui aprender bastante sobre as duas tecnologias. No entanto, acabei me desenvolvendo mais em React, que passou a ser parte constante nos projetos integradores seguintes, especialmente no desenvolvimento do frontend.
