Sistema de Análise de Interações com Processamento de Linguagem Natural para Geração de Insights em Atendimentos (2025.1)

Sobre o Projeto

Trabalho desenvolvido para o Projeto Integrador no 6º semestre da Fatec, tendo como cliente a empresa ITO1 Technology, especializada em desenvolver soluções personalizadas para as necessidades de seus clientes, incluindo softwares, hardwares e serviços inovadores.

Contribuições

Como projeto de encerramento do curso de Desenvolvimento de Software Multiplataforma da Fatec, aplicamos tudo o que aprendemos ao longo dos anos em um projeto com tema atual, envolvendo o uso de Processamento de Linguagem Natural (PLN), uma vertente da Inteligência Artificial. Com essa tecnologia, desenvolvemos um software capaz de analisar arquivos .csv de atendimentos, realizar o pré-processamento para limpeza dos dados e, em seguida, aplicar métodos de processamento, treinando um modelo para extrair informações úteis e gerar sumarizações. O sistema permitia, por exemplo, identificar soluções recorrentes para problemas similares. Por meio da exibição de dados como o número de chamadas classificadas como dúvidas, reclamações, entre outros, o software tornou-se uma ferramenta valiosa para gestores, revelando informações que antes estavam obscuras, por meio de um processo eficiente de Mineração de Dados.
